## Usar como extensión

Este repositorio puede ser añadido como una **extensión** en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **New Project**
* haga clic en **Extensiones** en el menú del engranaje
* buscar **https://github.com/oscarsanz2000/microbit-oscar-sanz** e importar

## Edita este proyecto ![Insignia de estado de compilación](https://github.com/oscarsanz2000/microbit-oscar-sanz/workflows/MakeCode/badge.svg)

Para editar este repositorio en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/oscarsanz2000/microbit-oscar-sanz** y haga clic en importar
